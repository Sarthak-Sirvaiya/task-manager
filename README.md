# Task Manager Backend 🚀

A full-stack Task Management API built using Node.js, Express, and MongoDB.

## 🔧 Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## ✨ Features
- Create a task
- Get all tasks
- Update task (mark complete, change priority)
- Delete task
- Task priority (low, medium, high)
- Due date support

## 📡 API Endpoints

### GET all tasks
GET /api/tasks

### CREATE task
POST /api/tasks

### UPDATE task
PUT /api/tasks/:id

### DELETE task
DELETE /api/tasks/:id
